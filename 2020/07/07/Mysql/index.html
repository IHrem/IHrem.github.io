备份数据库

1⃣️cd 保存路径

2⃣️mysqldump -uroot -p 数据库名 >导出的名字.sql

完成

恢复备份的数据库

1⃣️进入数据库

2⃣️选择数据库

3⃣️source .sql 文件的路径

完成

根据已有表建新表

1⃣️create table 新表名 like 旧表名

完成

修改操作

新建/删除列

  • 新建alter table tableName add column 名 type
  • 完成
  • 删除alter table tableName drop column 名
  • 完成

添加/删除主键

  • 添加alter table tableName add primary key(col)
  • 完成
  • 删除alter table tableName drop primary key(col)
  • 完成

修改表名

  • alter table 表名 rename to ‘新名’

增删改查

增

  • insert into 表名(col1,col2,…)values(val1,val2,…)

删

  • delete from 表名 where 条件

改

  • update 表名 set col=新数据 where col=旧数据

查

  • select * from 表名 where 条件

    • 推荐使用like语句select * from 表名 where file1 like ‘%value%’
    • 使用正则表达式推荐使用like语句select * from 表名 where col regexp ‘表达式‘

其他操作

排序(倒序/正序)

  • select * from 表名 order by col desc/asc

求和

  • select sum(col) as sum from 表名

均值

  • select avg(col) as average from 表名

最大

  • select max(col) as average from 表名

最小

  • select min(col) as average from 表名
